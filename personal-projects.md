## What I'm Building

Outside of work, I like to tinker. These are the projects I'm working on or thinking about — some practical, some just for fun.

---

### This Page :)

The site you're looking at. A Jekyll blog on GitHub Pages, managed by a custom Express-based admin panel I built with Kiro. It handles blog publishing, multi-platform syndication, resume management, media uploads, and AI-assisted proofreading — all from a single local UI. The whole thing is spec-driven with property-based tests.

---

### Pinewood Derby — DerbyNet on Raspberry Pi

Setting up [DerbyNet](https://derbynet.org/) on a Raspberry Pi to run Pinewood Derby races. Self-contained timing and scoring system that doesn't need an internet connection — just plug in the track sensors and go.

---

### Home SIP Server

Running a local SIP server at home for VoIP. After spending years working with enterprise telephony, it felt right to have my own setup. Useful for intercom, home office lines, and just understanding the protocol at a deeper level.
